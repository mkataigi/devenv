# Issue 分析スキル

GitHub Issue #$ARGUMENTS を分析し、技術仕様書を作成してください。

## 実行手順

1. GitHub API から Issue の詳細を取得:
```bash
# ヘルパースクリプトを使用して GitHub Issue を取得
./scripts/fetch-github-issue.sh $ARGUMENTS
```

2. 要件を十分に理解する
3. 関連するコードとプロジェクト構造をレビューする
4. 分析結果を明確に出力する
5. 以下のフォーマットで技術仕様書を作成する
